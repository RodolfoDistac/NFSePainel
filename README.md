# NFSe Painel (PySimpleGUI)

Painel visual para **ler, filtrar e exportar** NFS-e (XML ABRASF e variações).
- Entrada: **pasta**, **.zip** com XMLs, ou **.xml** único.
- Visualização em **tabela**, **filtro** em tempo real e **exportação CSV**.
- **Logs estruturados** (JSON) com **mascaramento de CPF/CNPJ**.
